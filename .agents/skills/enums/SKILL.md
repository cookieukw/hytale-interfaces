# Hytale UI Enums Reference

This folder contains detailed markdown documentation for the enum values used by properties in Hytale Custom UI.
If an element's property expects an enum (like `LayoutMode` or `LabelAlignment`), you can find all valid values and their behaviors by reading the corresponding markdown file in this directory.

## How to Use
- **Do not hallucinate enum values.** If you don't know the valid options for a property, check here.
- Use `list_dir` on `.agents/skills/enums/` to see all available enum documentation files.
- Use `view_file` on a specific enum (e.g., `layoutmode.md`) to read the exact valid strings you can use in `.ui` files.
