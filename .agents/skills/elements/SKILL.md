# Hytale UI Elements Reference

This folder contains detailed markdown documentation for each available UI element in Hytale.
When building or modifying `.ui` files and you need to know what properties an element accepts, or how it behaves, use the `view_file` tool to read the specific element's markdown file in this folder (e.g. `button.md`, `assetimage.md`).

## How to Use
- **Do not guess properties.** If you are unsure what properties an element supports, look it up here.
- Use `list_dir` on `.agents/skills/elements/` to see all available element files.
- Use `view_file` on a specific file (like `group.md` or `label.md`) to read its documentation and property tables.
