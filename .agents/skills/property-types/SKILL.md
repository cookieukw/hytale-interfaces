# Hytale UI Property Types Reference

This folder contains detailed markdown documentation for complex structural types used in Hytale Custom UI properties.
When you see a property in an element that takes a composite type like `Anchor`, `Padding`, `PatchStyle`, or `TextTooltipStyle`, read the corresponding markdown file here to understand its sub-properties and how to format them.

## How to Use
- **Check property structures.** Complex properties are written with parenthesis in Hytale UI, e.g., `Padding: (Left: 10, Right: 10)`. Look up the type here to see what sub-properties exist.
- Use `list_dir` on `.agents/skills/property-types/` to see all available type documentations.
- Use `view_file` on a specific type (e.g., `anchor.md` or `padding.md`) to read its documentation.
